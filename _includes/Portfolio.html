LMix

核心开发,视音及 CG 算法

基于 Unity3D(C#)以及新的体感设备 Leap Motion 开发,同时支持键盘或 Kinect。与普通音游不同的 是,游戏 供了一个 3d 空间而且玩家可以通过体感设备使用触碰和手势触击落键和旋转转盘,同时支 持多种难度, 添加歌曲,自定义谱面信息。该项目目前已在 Github 上开源并最终获得了 Leap Motion 开发大赛 第二名 。

体感游戏, MUG, Leap Motion, Unity3d, C#

Photo Theater | 照片剧院

一个云端照片管理网站，用户可以批量上传并管理图片，并通过地理位置，分类，标签，评论，心情，描述对照片进行查找和管理，并对照片进行滤镜添加和人像美化。团队自身开发出了项目中使用的十余种滤镜和美化算法，并提供了合理的图片滤镜运算缓存。因为管理和滤镜添加的便利，本人也在一些个人网站项目中使用这个平台作为图床。

Python, Django, LBS, 图片美化，人脸识别
